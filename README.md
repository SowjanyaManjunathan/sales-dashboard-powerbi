# Sales Data Analysis Dashboard (Power BI)

## 📊 Project Overview
Built an interactive Power BI dashboard analyzing retail sales across 45 stores to identify trends, store performance, and promotional impact.

## 🚀 Key Features
- Sales trend analysis over time
- Holiday vs Non-holiday performance
- Store type comparison
- Promotion impact analysis using scatter plots
- Interactive filters for store and date

## 🧠 Key Insights
- Sales show strong seasonal spikes during holiday periods
- Store Type A contributes highest revenue
- Promotions positively impact sales but vary across stores
- Stable baseline demand outside peak periods

## 🛠 Tools Used
- Power BI
- DAX
- Data Modeling
